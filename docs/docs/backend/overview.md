## **Tech Stack**

| Category | Technology | Description |
| -------- | ---------- | ----------- |
| **Framework** | [Fastapi](https://fastapi.tiangolo.com/) | Modern, fast (high-performance) web framework for building APIs |
| **Typification** | [Pydantic](https://docs.pydantic.dev/latest/) | Data validation and settings management using Python type annotations |
| **Database** | [PostgreSQL](https://www.postgresql.org/) | Powerful open source relational database system|
| **ORM** | [Sqlmodel](https://sqlmodel.tiangolo.com/) | ORM combining SQLAlchemy and Pydantic for type-safe database models |
| **Object storage**| [S3](https://aws.amazon.com/s3/) | Scalable cloud object storage service by AWS |
| **Analytics** | [Elastisearch](https://www.elastic.co/elasticsearch) | Distributed search and analytics engine for all types of data |
| **Cache** | [Redis](https://redis.io/) | In-memory key-value data store for caching and real-time applications |
| **Message broker** | [RabbitMQ](https://www.rabbitmq.com/) | Open source message broker for distributed systems |
| **CI/CD** | [GitHub Actions](https://docs.github.com/en/actions) | Automated workflows and CI/CD pipelining integrated with GitHub |
| **Auth** | [JWT](https://www.jwt.io/) + [OAuth 2.0](https://oauth.net/2/) | Secure authentication and authorization standards |
| **Testing** | [Pytest](https://docs.pytest.org/en/stable/) | Python testing framework for writing simple and scalable tests |
| **Containers** | [Docker](https://docs.docker.com/) | Containerization platform for applications and microservices |
| **Orchestration** | [Kubernetes](https://kubernetes.io/) | Container orchestration system for automating deployment, scaling |
| **Scraping** | [undetected-chromedriver](https://github.com/ultrafunkamsterdam/undetected-chromedriver) | Web scraping tool bypassing bot detection |
| **Marketing** | [GetResponse](https://www.getresponse.com/) | All-in-one online marketing platform |
| **Documentation**| [Sphinx](https://www.sphinx-doc.org/en/master/index.html) | Python documentation generator supporting reStructuredText |
